How scoring now works
  • Brand Match rewards actual VeriPura identity plus real system alignment.
  • Execution Quality measures tone, structure, metadata, and accessibility.
  • Candidate content is score-capped if identity proof is weak.
  • Reference content is judged more lightly on optional technical metadata.
